# CausaNet+ Architecture

## System Flow
Input Data → Work Graph Engine → Causal Engine → Decision Engine → Dashboard

## Components

### Work Graph Engine
Builds interaction network between users and identifies bottlenecks.

### Causal Engine
Analyzes cause-effect relationships in workflows.

### Decision Engine
Evaluates confidence vs accuracy of decisions.

### Verification Layer
- zkVerify → validates data integrity
- Horizon → stores immutable records
